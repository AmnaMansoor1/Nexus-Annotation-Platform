# Nexus Annotation Platform

This repository is organized into separate frontend and backend folders.

## Structure

- `frontend/`: React + TypeScript + Vite application for the annotation platform UI
- `backend/`: placeholder area for API, server, or automation code

## Commands

Run these from the repository root:

```bash
npm run dev
npm run build
npm run lint
```

Frontend-only commands:

```bash
npm run frontend:dev
npm run frontend:build
npm run frontend:lint
```

Backend scaffold command:

```bash
npm run backend:dev
```
